To run the code, you need these libraries
numpy
pandas
datetime
util.py
matplotlib.pyplot


1. Put all .py files in the same folder
2. To run indicators.py, simply do:
python indicators.py

It will generate 3 charts of the technical indicators I used in this study.
3. To run BestPossibleStrategy.py, simply do:
python BestPossibleStrategy.py

It will return a trade_df data frame, and generate graph and information for portfolio values.
4. To run ManualStrategy, simply do:
python ManualStrategy.py

It will return a trade_df data frame, and generate graph and information for portfolio values.
